Project Summary

The purpose of this project was to create a dashboard that visualizes and analyzes the correlation, if any, between a city's latitude and its weather.

Using data we pulled from the OpenWeatherMap API, we assembled a dataset on over 500 cities and created visualizations using Matplotlib to compare latitude to temperature, cloudiness, wind speed, and humidity.

After analyzing the data and looking at the resulting visualizations, only one weather factor can confidently be concluded to be correlated to a city's latitudinal coordinates. It is evident that the closer a city is to the equator, the warmer it is in max. temperature, whereas humidity, cloudiness, and wind speed are all factors that are not directly affected.
